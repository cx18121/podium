<objective>
Wire transcript persistence and live caption display across four files: schema migration (db.ts), storage (App.tsx), player prop (Review.tsx), and caption UI (AnnotatedPlayer.tsx).

Purpose: TranscriptSegment[] is already captured by SpeechCapture but thrown away after session save. Persisting it enables a live caption bar that syncs to video playback time, giving users a way to follow their own speech while reviewing the session.

</objective>

   function getCurrentCaption(

     const active = segments
