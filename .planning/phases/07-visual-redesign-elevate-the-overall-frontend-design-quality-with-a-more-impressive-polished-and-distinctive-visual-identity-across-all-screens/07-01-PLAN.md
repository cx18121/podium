The current `src/index.css` contains only `@import "tailwindcss";`. Replace the entire file contents with:

```css
@import "tailwindcss";

@layer base {
  :root {
    --color-bg: #080c14;
    --color-surface: #111827;
    --color-surface-raised: #1a2235;
    --color-accent: #6366f1;
    --color-accent-hover: #818cf8;
    --color-accent-glow: rgba(99, 102, 241, 0.15);
    --color-border: rgba(255, 255, 255, 0.07);
    --color-border-hover: rgba(255, 255, 255, 0.12);
    --color-border-modal: rgba(255, 255, 255, 0.10);
    --color-text-primary: #f1f5f9;
    --color-text-secondary: #94a3b8;
    --color-text-muted: #475569;
    --color-destructive: #ef4444;
  }

  body {
    background-color: var(--color-bg);
    color: var(--color-text-primary);
    font-family: 'Inter', system-ui, sans-serif;
  }
}
```

Keep `@import "tailwindcss";` as the FIRST line — Tailwind v4 requires it at the top. The `@layer base` block follows immediately after. Do NOT add `tailwind.config.js` or `postcss.config.js` — Tailwind v4 uses CSS-first configuration.
